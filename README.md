# HSI Classification Using State-of-the-Art Models

Thesis code for hyperspectral image classification experiments with multiple deep learning model variants.

## Repository contents
- `main_big.py`: experiment entry point
- `models.py`: primary model definitions
- `models_liu.py`: alternative model implementation
- `datasets.py`: dataset loading utilities
- `utils.py`: shared helpers
- `Ouzounidis_thesis[2086].pdf`: thesis document
- `LICENSE`: license file

## Notes
- This repository is oriented around research experiments rather than a packaged training framework.
- The thesis PDF provides the broader methodology and evaluation context for the code.
